# Misión Patrulla Verde

Objeto de Aprendizaje web interactivo basado en el cuento “La Patrulla Verde del Barrio”.

## Descripción

“Misión Patrulla Verde” es una página web educativa e interactiva pensada para estudiantes de primaria.

El recorrido acompaña a Sebastián, protagonista del cuento, en una misión para reconocer un problema ambiental en el barrio Villa Mar, tomar decisiones, organizar una Patrulla Verde, crear mensajes ambientales y asumir un compromiso personal de cuidado del ambiente.

## Objetivo

Que el estudiante pueda identificar un problema ambiental cercano, reconocer acciones concretas para resolverlo y formular un compromiso personal de cuidado del ambiente.

## Tipo de proyecto

Página web interactiva responsive.

Debe funcionar correctamente en:

- computadora;
- tablet;
- celular.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript vanilla

No requiere backend en esta primera etapa.

## Archivos principales

- `index.html`
- `styles.css`
- `script.js`
- `assets/images/`
- `assets/logos/`

## Estructura del recorrido

El Objeto de Aprendizaje se organiza en 10 pantallas:

1. Bienvenida a la misión.
2. Registro del estudiante.
3. Presentación del cuento.
4. Observación del problema.
5. Toma de decisión.
6. Armado de la Patrulla Verde.
7. Creación de frases ambientales.
8. Resultado de la misión.
9. Compromiso final.
10. Evidencia final descargable.

## Datos del estudiante

En esta primera versión, los datos del estudiante se utilizan únicamente dentro del navegador para personalizar la experiencia y generar la evidencia final.

No se envían datos a servidores externos.

## Estado del proyecto

Primera versión funcional preparada para publicación como sitio web estático.
