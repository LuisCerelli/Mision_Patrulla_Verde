# Mission Green Patrol

Interactive web-based Learning Object inspired by the story “The Green Patrol of the Neighborhood”.

## Description

“Mission Green Patrol” is an educational and interactive website designed for primary school students.

The experience follows Sebastián, the main character of the story, on a mission to identify an environmental problem in the Villa Mar neighborhood, make decisions, organize a Green Patrol, create environmental messages, and write a personal commitment to care for the environment.

## Objective

The goal of this Learning Object is for students to identify a nearby environmental problem, recognize concrete actions to help solve it, and express a personal commitment to environmental care.

## Project Type

Responsive interactive website.

It is designed to work properly on:

- desktop computers;
- tablets;
- mobile phones.

## Technologies Used

- HTML
- CSS
- Vanilla JavaScript

This first version does not require a backend.

## Main Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/images/`
- `assets/logos/`

## Learning Path Structure

The Learning Object is organized into 10 screens:

1. Mission welcome.
2. Student registration.
3. Story introduction.
4. Problem observation.
5. Decision-making.
6. Building the Green Patrol.
7. Creating environmental messages.
8. Mission result.
9. Final commitment.
10. Downloadable final evidence.

## Student Data

In this first version, student data is used only inside the browser to personalize the experience and generate the final evidence.

No student data is sent to external servers.
